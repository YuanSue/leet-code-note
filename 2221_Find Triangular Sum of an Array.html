Find Triangular Sum of an Array

Question

You are given a 0-indexed integer array nums, where nums[i] is a digit between 0 and 9 (inclusive).
The triangular sum of nums is the value of the only element present in nums after the following process terminates:

  1. Let nums comprise of n elements. If n == 1, end the process. Otherwise, create a new 0-indexed integer array newNums of length n - 1.
  2. For each index i, where 0 <= i < n - 1, assign the value of newNums[i] as (nums[i] + nums[i+1]) % 10, where % denotes modulo operator.
  3. Replace the array nums with newNums.
  4. Repeat the entire process starting from step 1.
    5. Return the triangular sum of nums.
    題意:
    • 給定一個 0-indexed（從 0 開始編號）的整數陣列 nums。
    • nums[i] 每個元素都是 0~9 之間的數字（單一位數）。
  1. Example 1:
    2. Input: nums = [1,2,3,4,5] Output: 8 Explanation: The above diagram depicts the process from which we obtain the triangular sum of the array.
  2. Example 2:
    3. Input: nums = [5] Output: 5 Explanation: Since there is only one element in nums, the triangular sum is the value of that element itself.

Tips

    步驟思路:
  1. 要計算「triangular sum（三角形總和）」，步驟如下：

Answer

    
from typing import List class Solution: def triangularSum(self, nums: List[int]) -> int: # 當陣列長度 > 1 就持續做 while len(nums) > 1: # 產生新的一層，長度少 1 nums = [(nums[i] + nums[i+1]) % 10 for i in range(len(nums) - 1)] # 最後只剩一個元素，直接回傳 return nums[0]

法二:

    
class Solution: def triangularSum(self, nums: List[int]) -> int: while len(nums) > 1: # 產生新的一層，長度少 1 new_nums = [] for i in range(len(nums)-1): new_nums.append((nums[i] + nums[i+1]) % 10) nums= new_nums # 最後只剩一個元素，直接回傳 return nums[0]

Explanation

nums = [(nums[i] + nums[i+1]) % 10 for i in range(len(nums) - 1)]其實就是「把 new_nums 的建立 + append 的動作合併在一起」的語法糖：
    • [...] 會直接產生一個新的 list
    • for i in range(...) 是迴圈
    • 每圈產生 (nums[i] + nums[i+1]) % 10 這個值，自動幫你放到 list 裡

所以它等同於：

    
new_nums = [] for i in range(len(nums)-1): new_nums.append((nums[i] + nums[i+1]) % 10) nums= new_nums
    這就是 list comprehension（串列生成式） 的好處：
    • 不用自己宣告 new_nums = []
    • 不用 append
    • 一行就完成「產生新串列」
    只要「一個迴圈 + 產生新值」，就能用 comprehension 取代整個 for 迴圈+append，程式碼更短、更直覺。
