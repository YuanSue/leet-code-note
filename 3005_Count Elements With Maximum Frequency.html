Maximum Number of Words You Can Type

Question

You are given an array nums consisting of positive integers.
Return the total frequencies of elements in nums such that those elements all have the maximum frequency.
The frequency of an element is the number of occurrences of that element in the array.

    題意:
    • 有一個鍵盤，其中有一些英文字母壞掉了（brokenLetters）。
    • text 是一個句子，由多個單字組成，單字之間只有一個空格。
    • 你要計算有幾個單字可以完全輸入（這個單字不包含任何壞掉的字母）。
  1. Example 1:
    2. Input: nums = [1,2,2,3,1,4] Output: 4 Explanation: The elements 1 and 2 have a frequency of 2 which is the maximum frequency in the array. So the number of elements in the array with maximum frequency is 4.
  2. Example 2:
    3. Input: nums = [1,2,3,4,5] Output: 5 Explanation: All elements of the array have a frequency of 1 which is the maximum. So the number of elements in the array with maximum frequency is 5.

Tips

    步驟思路:
  1. 頻率 (frequency)：某個數字在陣列中出現的次數。
  2. 先找出陣列裡出現次數最多的數字（也就是「最大頻率」）。
  3. 接著把所有擁有這個最大頻率的數字，它們在陣列中出現的次數加總起來。
  4. 回傳這個總和。

Answer

    
from typing import List from collections import Counter class Solution: def maxFrequencyElements(self, nums: List[int]) -> int: freq = Counter(nums) # 統計每個數字出現的次數 max_freq = max(freq.values()) # 找出最大頻率 return sum(c for c in freq.values() if c == max_freq) # 把所有達到最大頻率的次數加總
