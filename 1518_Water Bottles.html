Water Bottles

Question

There are water bottles (numBottles) that are initially full of water. You can exchange empty water bottles (numExchange) from the market with one full water bottle.
The operation of drinking a full water bottle turns it into an empty bottle.
Given the two integers numBottles and numExchange, return the maximum number of water bottles you can drink.

    題意 -> 你一開始有 numBottles 瓶水，全部都是滿的。規則是：
    • 喝掉一瓶滿的水 → 會得到一個空瓶。
    • 當你湊到 numExchange 個空瓶 → 可以去市場換 1 瓶新的滿水瓶。
      • 問：最多能喝掉多少瓶水？

Tips

    重點：
    • 初始答案：先把 numBottles 加進去，因為一開始就能喝光。
    • 迴圈模擬：
    • 當 numBottles >= numExchange，就能用空瓶換新水。
    • 每次換到新瓶，喝掉後又變成空瓶，可以繼續換。
    • 停止條件：當空瓶不夠再換 (numBottles < numExchange) 時結束。

Answer

    
class Solution: def numWaterBottles(self, numBottles: int, numExchange: int) -> int: drank = numBottles # 一開始先喝掉所有滿的 empty = numBottles # 所以得到這麼多空瓶 # 當空瓶還能換新水時 while empty >= numExchange: new_bottles = empty // numExchange # 能換到的新水瓶數 drank += new_bottles # 喝掉這些新瓶 empty = empty % numExchange + new_bottles # 剩下的空瓶數 return drank

執行範例

    
sol = Solution() print(sol.numWaterBottles(9, 3)) # 13 print(sol.numWaterBottles(15, 4)) # 19 print(sol.numWaterBottles(5, 5)) # 6

法二

    
class Solution: def numWaterBottles(self, numBottles: int, numExchange: int) -> int: return numBottles + (numBottles - 1) // (numExchange - 1)
