Water Bottles II

Question

You are given two integers numBottles and numExchange.
"numBottles" represents the number of full water bottles that you initially have. In one operation, you can perform one of the following operations:
Drink any number of full water bottles turning them into empty bottles.
Exchange "numExchange" empty bottles with one full water bottle. Then, increase numExchange by one.
Note that you cannot exchange multiple batches of empty bottles for the same value of numExchange. For example, if numBottles == 3 and numExchange == 1, you cannot exchange 3 empty water bottles for 3 full bottles.
Return the maximum number of water bottles you can drink.

    題意 -> 你一開始有 numBottles 個「滿水瓶」。每次操作可以做兩件事：
    • 喝掉一瓶滿水 → 變成一個空瓶。
    • 用 numExchange 個空瓶去換一瓶新的滿水瓶，但是換完後 numExchange 的需求會 +1（下一次要更多空瓶才能換）。
      • 目標：算出最多能喝多少瓶水。

Tips

    重點：
    • 用模擬的方式即可：
    • 記錄總共喝掉的瓶數。
    • 每次先喝光所有的滿水瓶 → 累加數量。
    • 用空瓶去檢查能不能換新瓶：
    • 如果 empty >= numExchange，就能換，空瓶數 -= numExchange，滿水瓶數 += 1，並且 numExchange += 1。
    • 重複直到不能換。
    • 回傳總數。

Answer

    
class Solution: def maxBottlesDrunk(self, numBottles: int, numExchange: int) -> int: drank = numBottles empty = numBottles n_bol = 0 while empty >= numExchange: empty = empty - numExchange n_bol +=1 numExchange += 1 empty += 1 drank += n_bol return drank

執行範例

    
sol = Solution() print(sol.maxBottlesDrunk(13, 6)) # 15 print(sol.maxBottlesDrunk(10, 3)) # 13 print(sol.maxBottlesDrunk(3, 1)) # 5
